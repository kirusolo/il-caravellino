# Il Caravellino – Digital Bar Menu

## Deploy to Vercel

1. Push this folder to a GitHub repository
2. Go to [vercel.com](https://vercel.com) → New Project
3. Import your GitHub repo
4. Vercel auto-detects Vite — just click **Deploy**
5. Your menu will be live at `your-project.vercel.app`

## Local Development

```bash
npm install
npm run dev
```

## Generate QR Code

Once deployed, go to [qr.io](https://qr.io) or [qrcode-monkey.com](https://qrcode-monkey.com), paste your Vercel URL and generate a QR code to print and place on tables!
